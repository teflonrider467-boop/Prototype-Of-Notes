The CSS applied in this paragraph is inline, i.e., CSS is written inside the tag itself, color is the PROPERTY and red is the VALUE, the style is applied only on the element it is written in

The CSS applied in this paragraph is internal, i.e., the element is targeted by specifying the tag itself or by specifying the ID(they have unique value and are used to uniquely identify elements) or CLASS(they don't have to be unique value and are used to identify elements in a group) assigned to it then the CSS is written inside STYLE tag inside the HEAD tag, the style is applied only on the element that are targeted in that particular page

The CSS applied in this paragraph is external, i.e., CSS is written in a different CSS file and then the file is linked to the HTML document using LINK tag, the style is applied to every page that has the CSS file is linked to it

The Inline CSS has the highest priority, followed by Internal CSS and then External CSS
